Projects

Website Kedai Kopi

"Kedai Kopi Senja Bercerita" is a web project designed to create a charming and informative online presence for a cozy coffee shop. This project is ideal for beginners and intermediate learners who wish to enhance their web development skills. The website features a clean and responsive design, ensuring a great user experience across different devices.

The key features of this project include:

  • Home Page: An inviting homepage with stunning visuals and a welcoming message to attract visitors.
  • Menu Section: A detailed menu showcasing the variety of coffee and food items available, complete with descriptions and prices.
  • About Us: A section that tells the story of "Kedai Kopi Senja Bercerita," highlighting the mission, vision, and values of the coffee shop.
  • Contact Page: An easy-to-use contact form for customers to reach out with inquiries, along with location details and opening hours.
  • Blog: A blog section where visitors can read articles about coffee culture, brewing techniques, and events happening at the coffee shop.
  • Responsive Design: The website is built to be fully responsive, providing an optimal viewing experience on desktops, tablets, and smartphones.

This project provides a comprehensive learning experience, covering essential web development concepts such as HTML, CSS, JavaScript, and responsive design techniques. By working on this project, learners will gain practical skills and confidence in creating functional and aesthetically pleasing websites.
